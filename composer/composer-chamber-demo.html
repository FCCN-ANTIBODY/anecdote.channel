The composer, running inside a chamber

The compose UI below lives in a powerless data: chamber (no crypto.subtle, no signing key, no trove, null origin). It can't reduce or sign anything itself — it summons those capabilities from this Elevated page over the probe line: label (Rung 0, live as you type) and sign-anecdote (Rung 1, on the confirmed Send). This page holds the reducer, the device key, and the trove; the chamber holds only the screen.
